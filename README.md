# 開発テンプレート集

プロジェクト開発で使える各種テンプレート・チェックリスト

## 📂 構成

- `code-review/` - コードレビューチェックリスト
- `project-setup/` - プロジェクトセットアップガイド（予定）
- `git-workflow/` - Git運用ルール（予定）

## 🚀 使い方

各テンプレートをプロジェクトにコピーして使用
```bash
# 例：レビューチェックリストをコピー
curl -O https://raw.githubusercontent.com/isyu-tsuji/dev-templates/main/code-review/REVIEW_CHECKLIST.md
```

## 📝 テンプレート一覧

### コードレビュー
- `REVIEW_CHECKLIST.md` - 汎用版チェックリスト
- `PYTHON_REVIEW.md` - Python特化版
- `API_REVIEW.md` - Web API特化版
